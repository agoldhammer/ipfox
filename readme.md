# commands

```bash
Usage: ipfox <COMMAND>

Commands:
  ips     List unique ips for database
  logs    List hostdata and logentries associated with given ip address
  all     Output logentries for each ip in hostdata
  counts  Output count of logentries for each ip in hostdata
  del     Delete ips and associated logentries
  help    Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
